# Live Demo

The credit card fraud detection model is deployed as a Gradio app on Hugging Face Spaces.

Try it here:  
https://huggingface.co/spaces/aminano/credit-card-fraud-detection-Gradio

## About the demo

The Space has two views:

- **Detailed UI (for display)** – a nicer-looking version with some extra fields that are just for show.
- **Simple Model Interface** – a clean version that only uses the inputs the model was actually trained on.
